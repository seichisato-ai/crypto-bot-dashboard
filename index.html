トークンの作り方
  • GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  • Repository access: Only select repositories → bot のリポジトリだけを選択
  • Permissions (Repository): Contents: Read-only / Actions: Read and write / Variables: Read and write
  • 生成された github_pat_... を上に貼り付け
